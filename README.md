# Model Rocketry Society Website

A premium, galaxy-themed website for the Model Rocketry Society, designed to attract younger students (Grade 6-12) to aerospace and rocketry.

## Project Structure

- `index.html`: The main homepage (including Membership & Benefits).
- `gallery.html`: Showcase for projects, sessions, events, and Board of Directors.
- `style.css`: All visual styling (Galaxy Classy theme).
- `app.js`: Interactions, animations, and form handling.
- `assets/`: Contains images and brand assets.

## How to Run

### Option 1: Direct File Open
Simply double-click `index.html` to open it in your browser.

### Option 2: Local Server (Recommended)
If you have Python installed, you can run a local server to see the site exactly as it would appear online.

1.  Run the helper script:
    ```bash
    python run_server.py
    ```
2.  Open `http://localhost:8000` in your browser.

## Deployment
To publish this website online:
1.  **GitHub Pages**: Upload these files to a GitHub repository and enable Pages.
2.  **Netlify/Vercel**: Drag and drop this folder onto their dashboard.
